建立新的分`支：
```
git branch fixbug
```ß

切换分支：
```
git checkout fixbug
```

上面两条命令可以合并为：
```
git checkout -b fixbug master
```

合并分支：
```
git checkout master
git merge fixbug
```

删除分支
```
git branch -d fixbug
```
